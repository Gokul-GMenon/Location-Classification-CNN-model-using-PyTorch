# A location Classification CNN model using PyTorch
This is CNN model built using PyTorch which aims at classifying images between forests, glaciers, buildings, mountains, seas and streets. The dataset contains around 2000 images in each of the categories adding to a total of 14034 images.

# Model
* Multiclass classification model
* Framework: PyTorch version 1.10.0
* Architecture: Convolutional Neural Network 2D
